# ethereum-canvas

## How To
Dependencies
```
$ npm install -g truffle
$ npm install -g ethereumjs-testrpc
```
Compile contract
```
$ truffle compile
```
Deploy/migrate contract
```
$ truffle migrate
```
Run local ETH blockchain
```
$ testrpc
```
Run app
```
$ npm run start
```
Build app
```
$ npm run build
```
